# 📖 Руководство пользователя - Informnews.kz

**Полное руководство по работе с новостным порталом**

---

## 🌐 О проекте

**Informnews.kz** - современный двуязычный (казахский/русский) новостной портал. Включает публичный сайт для читателей и административную панель (CMS) для управления контентом.

### Основные возможности:
- ✅ Публикация новостей на 2 языках
- ✅ Редактор как в Microsoft Word (TinyMCE)
- ✅ Автоматическая обработка изображений
- ✅ Модерация комментариев
- ✅ Поиск по новостям
- ✅ Виджеты погоды и курсов валют
- ✅ Защита от роботов (reCAPTCHA)

---

## 🔐 Вход в админ-панель

### Шаг 1: Открыть страницу входа

В браузере откройте:
```
https://ваш-сайт.kz/admin/login
```

### Шаг 2: Ввести данные

![Login Form Screenshot]

Вы увидите форму входа:
- **Логин** - введите ваше имя пользователя
- **Пароль** - введите пароль

Нажмите кнопку **"Кіру / Войти"**

## 📝 Управление новостями

### Просмотр всех новостей

1. Нажмите **"Мақалалар / Посты"** в левом меню
2. Увидите список всех новостей:
   - Заголовок
   - Категория
   - Статус (Опубликовано/Черновик)
   - Дата создания
   
### Создание новой новости

#### Шаг 1: Открыть форму

1. В разделе "Мақалалар" нажмите **"+ Жаңа мақала қосу"**
2. Откроется редактор новости

#### Шаг 2: Заполнить заголовок

Вверху видите 2 вкладки:
- **🇰🇿 ҚАЗАҚША (KZ)** - казахская версия
- **🇷🇺 РУССКИЙ (RU)** - русская версия

Заполните заголовок на обоих языках:

**На казахском:**
```
Заголовок (QAZ): Алтын бағасы алғаш рет 5300 доллардан асты
```

**На русском:**
```
Заголовок (RUS): Цена золота впервые превысила 5300 долларов
```

> **Совет:** Минимум нужен заголовок на 1 языке, но лучше заполнить оба!

#### Шаг 3: Написать текст новости

В каждой вкладке есть **большой редактор** (как в Word!).

**Возможности редактора:**

1. **Форматирование текста:**
   - **Жирный** - выделить текст и нажать кнопку B
   - *Курсив* - кнопка I
   - Подчеркнутый - кнопка U
   - Зачеркнутый - кнопка S

2. **Заголовки:**
   - Выделите строку
   - Выберите в выпадающем списке "Heading 2" или "Heading 3"

3. **Списки:**
   - Нумерованный список - кнопка 1. 2. 3.
   - Маркированный список - кнопка • • •

4. **Вставка изображений в текст:**
   - Нажмите кнопку 🖼️ (картинка)
   - Выберите файл с компьютера
   - Картинка вставится прямо в текст!

5. **Вставка ссылок:**
   - Выделите текст
   - Нажмите кнопку 🔗 (цепочка)
   - Вставьте URL адрес

6. **Таблицы:**
   - Нажмите кнопку таблицы
   - Выберите размер (строки x столбцы)
   - Заполните ячейки

**Пример хорошей новости:**
```
Қызылордада алтын бағасы тарихи максимумға жетті

Бүгін халықаралық нарықта алтын бағасы алғаш рет 1 унциясы үшін 
5300 доллардан асты. Қаржы сарапшылары мұны...

[Здесь можно вставить фото графика]

Негізгі себептер:
• Әлемдік экономикалық тұрақсыздық
• Доллардың әлсіреуі  
• Инвесторлардың сұранысы

Детали в полной версии...
```

#### Шаг 4: Установить время публикации

Справа найдите блок **"⏰ Уақыты / Время публикации"**:

- **Вариант 1:** Оставьте пустым - новость опубликуется прямо сейчас
- **Вариант 2:** Выберите дату и время - новость выйдет в это время

> **Важно:** Новости сортируются по этому времени! Если хотите чтобы новость была выше - поставьте более позднее время.

#### Шаг 5: Загрузить главное фото

Справа в блоке **"Басты сурет / Главное фото"**:

1. Нажмите **"Жүктеу үшін басыңыз"** (или просто кликните по серой зоне)
2. Выберите фото с компьютера
3. **Сразу увидите превью!** 

Требования к фото:
- ✅ Форматы: JPG, PNG
- ✅ Макс. размер: 5 МБ
- ✅ Рекомендуемый размер: 1200x800 пикселей

> **Система автоматически создаст 3 версии:**
> - Маленькую (thumbnail) - 150x150
> - Среднюю (medium) - 600x400  
> - Большую (large) - 1200x800

#### Шаг 6: Выбрать категорию

Справа найдите **"КАТЕГОРИЯ"**:

Отметьте одну из категорий:
- ☑️ Жаңалықтар (Новости)
- ☑️ Саясат (Политика)
- ☑️ Қоғам (Общество)
- ☑️ Экономика
- ☑️ Білім (Образование)
- ☑️ Спорт

> **Категорию обязательно нужно выбрать!**

#### Шаг 7: Опубликовать или сохранить

Вверху справа и справа в блоке есть кнопки:

**Вариант 1: Опубликовать сразу**
- Нажмите красную кнопку **"Жариялау / Опубликовать"**
- Новость сразу появится на сайте!

**Вариант 2: Сохранить черновик**
- Нажмите серую кнопку **"Сақтау / Сохранить"**
- Новость сохранится, но НЕ появится на сайте
- Позже можете отредактировать и опубликовать

#### Итого: Быстрая инструкция

```
1. "Мақалалар" → "+ Жаңа мақала қосу"
2. Введите заголовок (KZ и RU)
3. Напишите текст в редакторе  
4. Вставьте фото в текст (если нужно)
5. Загрузите главное фото
6. Выберите категорию
7. Нажмите "Жариялау"
✅ Готово!
```

---

### Редактирование существующей новости

1. Откройте **"Мақалалар / Посты"**
2. Найдите нужную новость в списке
3. Нажмите кнопку **"Өзгерту / Редактировать"** (✏️)
4. Внесите изменения
5. Нажмите **"Сақтау"** или **"Жариялау"**

> **Важно:** При редактировании вы можете изменить время публикации - новость переместится в списке по новому времени!

---

### Удаление новости

1. Откройте список новостей
2. Найдите новость
3. Нажмите красную кнопку **"Жою / Удалить"** (🗑️)
4. Подтвердите удаление

> **Внимание:** Удаление нельзя отменить! Будьте осторожны.

## 📁 Управление категориями

### Просмотр категорий

1. В левом меню нажмите **"Санаттар / Категории"**
2. Увидите список всех категорий:
   - Название (на обоих языках)
   - Количество постов
   - Дата создания

### Создание новой категории

1. Нажмите **"+ Жаңа санат қосу / Добавить категорию"**
2. Заполните поля:
   - **Название (KZ):** Например "Денсаулық"
   - **Название (RU):** Например "Здоровье"
   - **Описание (KZ):** Краткое описание категории
   - **Описание (RU):** Краткое описание
3. Нажмите **"Сақтау / Сохранить"**

### Редактирование категории

1. В списке категорий нажмите **"Өзгерту"** (✏️)
2. Измените данные
3. Сохраните

### Удаление категории

> **Внимание:** Нельзя удалить категорию, если в ней есть новости!

---

## 💬 Модерация комментариев

### Просмотр комментариев

1. Откройте **"Пікірлер / Комментарии"**
2. Увидите список:
   - Текст комментария
   - Автор
   - К какой новости
   - Статус (Ожидает/Одобрено)
   - Дата

### Одобрение комментария

1. Найдите комментарий со статусом **"Ожидает модерации"**
2. Нажмите зеленую кнопку **"✓ Бекіту / Одобрить"**
3. Комментарий появится под новостью на сайте

### Удаление комментария

1. Найдите нежелательный комментарий
2. Нажмите красную кнопку **"🗑️ Жою / Удалить"**
3. Подтвердите

> **Совет:** Удаляйте спам, оскорбления и неуместные комментарии

---

## 👥 Управление пользователями (администраторами)

### Просмотр администраторов

1. Откройте **"Қолданушылар / Пользователи"**
2. Увидите список всех админов

### Создание нового администратора

1. Нажмите **"+ Жаңа қолданушы / Добавить пользователя"**
2. Заполните:
   - **Логин:** имя для входа (латиницей)
   - **Email:** почта
   - **Полное имя:** ФИО
   - **Пароль:** надежный пароль
   - **Роль:** Admin или Editor
3. Нажмите **"Сақтау"**

### Изменение пароля

1. Откройте пользователя на редактирование
2. В поле **"Новый пароль"** введите новый
3. Сохраните

## ⚙️ Настройки сайта

### Открыть настройки

1. В левом меню нажмите **"Баптаулар / Настройки"**
2. Откроется страница с настройками

### Что можно настроить:

#### 1. Контактная информация
- **Email редакции:** для связи
- **Телефон:** будет показан на сайте
- **Адрес:** физический адрес редакции

#### 2. Социальные сети
Укажите ссылки на ваши профили:
- Facebook
- Instagram  
- Twitter
- YouTube
- Telegram

> **Совет:** Вставляйте полные ссылки, например:
> `https://instagram.com/informnews`

#### 3. API ключи

**OpenWeather API (погода):**
1. Зарегистрируйтесь на https://openweathermap.org
2. Получите бесплатный API ключ
3. Вставьте в поле **"OpenWeather API Key"**

**Google reCAPTCHA (защита от ботов):**
1. Откройте https://www.google.com/recaptcha/admin
2. Создайте Site (v2 Checkbox)
3. Скопируйте **Site Key** и **Secret Key**
4. Вставьте в соответствующие поля

#### 4. Сохранить изменения

После всех изменений нажмите **"Сақтау / Сохранить"** внизу страницы.

## 🔍 Функция поиска

Пользователи могут искать новости на сайте.

### Как пользователи ищут:

**Вариант 1:** Через header
- В шапке сайта есть форма поиска
- Вводят запрос
- Нажимают Enter

**Вариант 2:** Через кнопку лупы
- Кликают на иконку 🔍 в правом верхнем углу
- Откроется модальное окно
- Вводят запрос
- Нажимают "Іздеу / Найти"

**Вариант 3:** Горячая клавиша
- Нажимают `Ctrl+K` (или `Cmd+K` на Mac)
- Открывается поиск

### Что ищется:
- Заголовки новостей
- Текст новостей
- Отрывки (excerpt)

## 📱 Публичный сайт

### Страницы сайта:

1. **Главная** (`/`)
   - Hero новость (самая свежая)
   - 6 последних новостей

4. **Поиск** (`/search?q=запрос`)

В правом верхнем углу кнопка **KZ / RU**:
- Текущий язык подсвечен красным
- Клик переключает язык
- Сайт переводится полностью

## ✅ Правила публикации

### Хорошая новость содержит:

1. ✅ **Заголовок на обоих языках**
   - Краткий (до 100 символов)
   - Цепляющий
   - Отражает суть

2. ✅ **Текст от 200 символов**
   - Структурированный
   - С подзаголовками  
   - Разбит на абзацы

3. ✅ **Главное фото высокого качества**
   - Четкое, не размытое
   - Горизонтальное (landscape)
   - Размер мин. 1200x800

4. ✅ **Правильная категория**

5. ✅ **Актуальное время публикации**

### Чего избегать:

- ❌ Заголовок только на 1 языке
- ❌ Слишком короткий текст
- ❌ Плохое фото (низкое качество)
- ❌ Грамматические ошибки
- ❌ Не выбрана категория

## 🛡️ Безопасность

### Важные правила:

1. **Никому не давайте пароль от админки**
2. **Используйте надежный пароль:**
   - Минимум 8 символов
   - Буквы + цифры + символы
   - Пример: `Admin2026!Inf`

3. **Регулярно меняйте пароль** (раз в 3 месяца)

4. **Выходите из системы:**
   - После работы нажимайте **"Шығу / Выйти"** в левом меню

5. **Проверяйте reCAPTCHA:**
   - При входе обязательно ставьте галочку "Я не робот"

## 📞 Помощь и поддержка

### Если что-то не работает:

1. **Обновите страницу** (F5 или Ctrl+R)
2. **Очистите кеш браузера** (Ctrl+Shift+Del)
3. **Попробуйте другой браузер** (Chrome, Firefox)

### Частые проблемы:

**Проблема:** Не загружается фото
**Решение:** 
- Проверьте размер (макс. 5 МБ)
- Проверьте формат (JPG или PNG)
- Попробуйте другое фото

**Проблема:** Редактор не работает
**Решение:**
- Обновите страницу
- Попробуйте в режиме инкогнито

**Проблема:** Новость не появляется на сайте
**Решение:**
- Проверьте что нажали "Жариялау" (Опубликовать)
- Проверьте что время публикации не в будущем
- Обновите главную страницу сайта (Ctrl+F5)

## 📚 Глоссарий

**Термины которые нужно знать:**

- **CMS** - система управления контентом (админ-панель)

- **Thumbnail** - маленькая версия картинки (превью)
