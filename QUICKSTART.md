# QUICK START - Informnews.kz

## 🚀 Запуск за 4 шага

### Шаг 1: Установка зависимостей

Установите необходимые PHP расширения:
```bash
# PHP GD для обработки изображений
sudo apt-get install -y php8.3-gd

# Перезапустите PHP (если нужно)
sudo systemctl restart php-fpm  # или apache2
```

### Шаг 2: Настройка БД

Запустите скрипт автоматической настройки:
```bash
./setup-database.sh
```

Или вручную:
```bash
# Войти в MySQL
sudo mysql -u root

# Создать БД
CREATE DATABASE informnews CHARACTER SET utf8mb4;
USE informnews;

# Импортировать схему
SOURCE database/schema.sql;

# Импортировать демо-данные (опционально)
SOURCE database/sample_data.sql;

# Выйти
EXIT;
```

### Шаг 3: Конфигурация

Проверьте настройки в `config/database.php`:
```php
'host' => 'localhost',
'database' => 'informnews',
'username' => 'informnews_user',
'password' => 'informnews_pass_2026'
```

### Шаг 4: Запуск сервера

```bash
cd public
php -S localhost:8000
```

**Откройте в браузере:** http://localhost:8000

## 📸 Работа с изображениями

При создании/редактировании новости:

1. **Главное фото** - загружается в правой колонке
2. **Изображения в тексте** - вставляются через CKEditor:
   - Кнопка 🖼️ в toolbar
   - Drag & Drop прямо в редактор

Все картинки автоматически:
- ✅ Ресайзятся в 3 размера (thumbnail, medium, large)
- ✅ Сохраняются в `/public/uploads/`
- ✅ Оптимизируются для веба

## 🎯 Основные фичи

### Автоматический Hero
- Самая свежая новость всегда в "Басты жаңалық"
- Сортировка по дате публикации
- Не нужно вручную выбирать

- Форматирование как в Word
- Жирный, курсив, заголовки
- Списки, таблицы, цитаты
- Вставка ссылок и изображений
- 100% бесплатный, без API ключа

### Превью загруженных фото
- Видно сразу после выбора файла
- Кнопка удаления для перевыбора
- Не нужно сохранять форму
